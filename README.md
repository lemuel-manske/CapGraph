# CapGraph

This project aims to explore the hability to enable LLM agents to navigate and utilize a graph of capabilities in the form of tools,
enabling them to solve problems by navigating a graph of capabilities instead of blowing the context window with all tools at once, thus
using a graph-constrained engine that releases tools by demand.

## Motivation

Create a more efficient way for LLMs to utilize tools/workflows/capabilities by providing them on demand, instead of overwhelming them with a large number of tools at once.

What it is not about:
- Creating a new graph database
- Creating a new model
- Fine-tunine a new model
